# Blog Project

## Overview

A blog platform project located at `blog.thepaynetrain.com`.

## Getting Started

1. Clone the repository
2. Install dependencies
3. Configure your environment
4. Run the development server

## Project Structure

```
/
├── README.md
├── src/
├── public/
└── config/
```

## Features

- Blog post management
- Responsive design
- Content publishing

## Development

```bash
npm install
npm run dev
```

## Deployment

Deploy to your hosting platform by running:

```bash
npm ci
npm run clean
npm run build
```

Then deploying a static site to your provider of choice
